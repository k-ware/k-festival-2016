N-Queen 퍼즐 설명 - 1

N-Queen 퍼즐은 N x N 크기의 체스판에 N 개의 퀸을, 서로 공격할 수 없도록 올려놓는 퍼즐이다.
(퀸은 체스에서 가장 강력한 기물로, 자신의 위치에서 상하좌우, 그리고 대각선 총 8방향으로 이어진 직선 상의 어떤 기물도 공격할 수 있다)

예를 들어, 위의 그림은 8x8 크기의 체스판에 8개의 퀸을 서로 공격할 수 없도록 올려놓은 예를 보인다.

체스판의 크기 N 이 주어졌을 때, N-Queen 퍼즐의 답이 모두 몇 개나 되는지를 계산하는 프로그램을 작성하시오.
한 답은 N개 퀸 모두의 위치로 정의되며, 한 퀸의 위치만 다르더라도 다른 답이라고 가정한다.

정답 제출

실행 코드의 클래스명은 Main이며 public static void main(String[] args) 메서드가 있어야 합니다.
또한 패키지를 지정해서는 안됩니다.
컴파일 및 실행 환경은 Java 8입니다.

이벤트 기간 및 상품

이벤트 기간은 2월 29일부터 3월 7일 오전 9시까지 진행했습니다.

각 실행이 서로 간섭이 없음을 확인해서, 실행 시간을 그대로 결정합니다.
아래 최종 결과에서 소스코드를 확인 할 수 있습니다.

등수는 가장 짧은 실행 시간에 정답을 출력하는 사람이 1등입니다.
회사에는 다양한 경력의 사람들이 있어서 개발 경력을 고려하여 과장 이상과 미만으로 그룹을 나누었습니다.

각 그룹별로 1등에게 상품을 지급합니다.
단 과장 이상 그룹의 1등이 미만 그룹 1등 보다 빠르지 않은 경우, 미만 그룹의 1등에게만 상품을 지급합니다.

이번 이벤트가 전체 상금 100만원 이내에서 진행하는 이벤트이며,
각 그룹의 1등에게 50만원 상당의 상품을 제공할 예정입니다.
각 그룹 1등 수상자가 원하는 물건을 정하면, 회사에서 구매해 드립니다. 금액 추가분에 대해서는 개인이 부담할 수 있습니다.

상품 증정에는 조건이 있습니다. 문제가 인터넷에서 쉽게 답을 찾을 수 있으므로 문제 풀이(알고리즘 설명)를 회사 블로그에 게시한 후 수령 가능합니다.
아마도 처음으로 발행되는 내부작성 컨텐츠가 될것 같습니다.
